# Análise Financeira de Loja em Python

Projeto desenvolvido para simular uma análise financeira simples de uma loja, aplicando conceitos fundamentais de lógica de programação.

## 🔧 Funcionalidades

- Atualização de faturamento com novas vendas
- Cálculo de imposto sobre o faturamento
- Cálculo de lucro líquido
- Cálculo da margem de lucro
- Identificação se houve lucro (booleano)
- Conversão de tempo (meses → anos e meses)

## 💡 Conceitos aplicados

- Operações matemáticas
- Variáveis e tipos de dados
- Formatação de saída (f-string)
- Operadores lógicos
- Divisão inteira (//) e módulo (%)

## ▶️ Como executar

```bash
python main.py
